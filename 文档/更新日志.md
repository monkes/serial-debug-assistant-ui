- ✨ 实现日志窗口和调试输出窗口的自动滚动

- ✨ 支持从目录批量加载插件
- ✨ 支持从文件加载单个插件
- ✨ 实现插件激活和停用机制
- ✨ 提供插件权限管理功能

### 技术细节

#### 依赖项
- Python 3.6+
- PyQt5 5.15.0+
- PySerial 3.5+

#### 目录结构
```
debug-assistant/
├── debug_assistant/          # 主程序目录
│   ├── main.py              # 程序入口
│   ├── main_window.py       # 主窗口实现
│   ├── plugin_interface.py  # 插件接口
│   ├── plugin_manager.py    # 插件管理器
│   ├── plugins/            # 系统插件目录
│   └── resources/          # 资源文件
├── debian/                  # Debian打包配置
├── 文档/                    # 项目文档
└── share/                   # 共享资源
```

#### 插件接口
- `get_name()`: 获取插件名称
- `get_description()`: 获取插件描述
- `get_version()`: 获取插件版本
- `get_author()`: 获取插件作者
- `initialize(main_window)`: 初始化插件
- `activate()`: 激活插件
- `deactivate()`: 停用插件

## 版本说明

### 版本号格式
采用语义化版本控制（Semantic Versioning）：
- 主版本号：不兼容的API修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正
